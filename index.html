fka vRLI Sizer · EPS-Based Log Infrastructure Sizing

Sizing Scope

// Design & Operational Notes
📋 PnP Workbook is Mandatory
The VCF Planning & Preparation Workbook must be completed alongside sizer outputs before deployment. Sizer tools provide capacity estimates — the PnP Workbook drives the actual deployment configuration.
📥 vSAN Sizer Input Sources
Accepts LiveOptics (recommended for real-world I/O accuracy), RVTools (VM inventory export), or manual input. LiveOptics provides per-VM I/O profiling for the most precise cluster sizing.
💾 Logs Disk Sizing Baseline
~1 TB/node per 30-day retention at 5,000 EPS. Scale to ~3 TB/node for 90-day retention at equivalent EPS. Actual consumption varies significantly with log verbosity and source count.
📈 Size for Projected Growth
VCF Operations HA mode improves fault tolerance — it does not increase metric ingestion capacity. Configure the growth multiplier in the sizer and provision headroom for fleet expansion beyond the initial deployment scope.
